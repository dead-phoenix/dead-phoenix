### 👗 Australian Fashion Trends & Economic Indicators *(2010–2024)*
A data-driven investigation into whether economic signals shape fashion adoption — empirically testing the famous "Hemline Index."
- **Stack:** Python, R, Google Trends API, ABS data
- **Highlights:** Analysed 14 years of data using Granger causality and lag analysis; found a **0.86 correlation** between inflation and fast fashion search interest

---

### 📉 Behavioral Stress Index (BSI): Economic Forecasting
Developed a composite economic stress index from high-frequency search data — designed to detect economic shocks *faster* than traditional indicators.
- **Stack:** Python, PCA, Ridge Regression, walk-forward validation
- **Highlights:** Achieved **0.731 ROC AUC** in stress event classification; outperforms lagging macro indicators
